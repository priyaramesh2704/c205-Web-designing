Topic Description
Program A program is a set of instructions that directs a computer to perform specific tasks.
Algorithm An algorithm is a step-by-step procedure for solving a problem or performing a task.
Flowchart A flowchart is a visual diagram that represents the steps and decision points of a process or algorithm.
Compiler A compiler is a software tool that translates code written in a high-level programming language into machine code executable by a computer.
Interpreter An interpreter is a program that executes code line-by-line, translating high-level language into machine code in real time.
